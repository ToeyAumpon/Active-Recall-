# Active Recall 

## 1. システムの説明

React Nativeで構築されたモバイル向けフラッシュカードアプリです。
間隔反復（Spaced Repetition）を利用し、ノートを読み返すよりも「早く」「定着させる」学習を支援します。

---

### 誰に向けたものか?

従来の学習方法（マーカーを引く、読み返す、要約するなど）よりも効率的に学習したい学生や独学者。

**重要な前提条件:** 学習者は既に教材を読み、理解していること。このアプリは初めて学習する人向けではなく、既に理解している知識を*定着させ、記憶に留める*ためのものです。

---

## 2.システム概要

Active Recallは、受動的な学習（読み返し、ハイライト）をアクティブ・リコール（能動的想起）に置き換えるアプリです。情報を単に再読するのではなく、記憶から引き出すという科学的に証明された手法を採用しています。

このアプリの核心は「思い出すことに成功するたびに記憶が強化され、失敗すればそのカードはすぐに再提示される」という点にあります。時間が経つにつれ、覚えているカードの復習時間は減り、苦手なカードに自動的に集中できるようになります。

## 3.機能一覧

1.認証

メール・パスワードによるログイン/新規登録?
セッション維持とログアウト機能
再ログイン時のデータ復元（カード、継続日数の同期）

2.ホーム

本日の復習数: 今日やるべき枚数を表示
継続カウンター: 連続学習日数を強調表示
プッシュ通知: ユーザーが設定した時間にリマインダーを送信

問題作成 

テキスト入力: 表（問題）と裏（答え）を手動作成
画像入力: 写真や図解をカードに使用
トピック管理: カテゴリ分け、編集、削除

学習モード

カードめくり: タップして回答を表示
評価ボタン: 「Again（もう一度）」「Hard（難しい）」「Good（正解）」の3段階
スワイプ操作: 左＝Again、右＝Good、上＝Hardの高速操作

### 認証

| 機能 | 説明 |

|---------|-------------|

| メールアドレスとパスワードによるログイン | 標準的な認証情報による認証 |

| サインアップ | 新規アカウント作成 |

| セッションの維持 | アプリを再起動してもログイン状態を維持 |

| ログアウト | ローカルセッショントークンをクリア |


### ホーム

| 機能 | 説明 |

|---------|-------------|

| 学習期限カード数 | 本日復習が必要なカードの枚数を正確に表示 |

| 復習間隔の概要 | アクティブなSRSサイクル（1日、3日、1週間、30日）を表示 |

| 連続学習日数カウンター | 現在連続して学習した日数を分かりやすく表示 |

|クイックナビゲーションボタン | ワンタップで問題生成、学習モード、進捗状況へ |

| 初期設定 | 初めてご利用の方には、最初のカード作成方法をご案内します |

| 毎日のプッシュ通知 | ユーザーがリマインダー時間を設定し、通知で期限回数を表示します |

### 問題生成

| 機能 | 説明 |

|---------|-------------|

| テキスト入力 | 質問（表）と回答（裏）を手動で入力します |

| 画像入力 | 写真、スクリーンショット、図をカードの表または裏としてアップロードします |

| トピックタグ付け | 各カードをトピック/科目に割り当ててフィルタリングします |

| カード一覧表示 | 特定のトピックに関する既存のカードをすべて閲覧します |

| カード編集 | 既存のカードの誤字脱字を修正したり、コンテンツを更新したりできます |

| カード削除 | カードを完全に削除します |

| 画像圧縮 |アップロード前に画像は最大2MBに圧縮されます |

### 学習モード

| 機能 | 説明 |

|---------|-------------|

| カードめくり | タップして解答面を表示 |

| 復習モード | 1枚ずつカードを表示 — 問題 → 復習 → 表示 → 評価 |

| 説明モード | トピック名のみが表示され、ユーザーは記憶に基づいてすべてを説明し、その後すべてのカードを表示して自己チェックを行います |

| 評価ボタン | 各ボタンに「もう一度」「難しい」「良い」のプレビューが表示され、次の復習間隔がプレビューされます |

| スワイプ操作 | 左にスワイプ = 「もう一度」、右にスワイプ = 「良い」、上にスワイプ = 「難しい」（ボタンよりも高速） |

| スケジュール | 各評価により、カードの難易度と次回の復習日が自動的に更新されます |

| 進捗カウンター | 現在のカードの位置が表示されます（例：「8 / 18」） |

| セッション完了画面 | 期限が過ぎたカードがすべてクリアされると、正答率、復習済みカード数、連続記録の更新が表示されます | ### 進捗状況

###　進捗

| 機能 | 説明 |

|---------|-------------|

| 連続学習カレンダー | 学習日、欠席日、今日の日付を表示する14日間のグリッド |

| トピック習熟度バー | トピックごとの進捗状況を「習熟済み」「学習中」「新規」に分類 |

| 復習スケジュール | 各期間（1日・3日・1週間・30日）ごとにキューに追加されるカード |

| 週間ボリュームチャート | 過去7日間の1日あたりの復習カード数を棒グラフで表示 |

| 正答率 | 全復習における総合的な復唱正答率 |
